2. Задание для самоподготовки
1.1 Цель
  Создать адаптивную веб  страницу с использованием Flexbox и media query.

1.2 Описание
  В качестве основы можно использовать проект из предыдущих лабораторных работ.
  При выполнении задание необходимо учесть требования указанные ниже:
   Проект должен содержать несколько новых страниц в виде отдельных html файлов:
     Главная;
      Проекты;
      О приложении.
    Необходимо добавить favicon на все страницы веб-приложения.
    Каждая страница должна иметь корректный заголовок отображающийся во вкладке браузера.
    На каждую страницу необходимо добавить на страницы шапку-меню с несколькими пунктами-ссылками, которые ведут на соответствующие страницы:
  Главная;
  Проекты;
  Список задач (создана в ходе предыдущих лабораторных работ);
  О приложении.
  При нахождении на странице, соответствующий пункт меню должен быть выделен отличным от других пунктов стилем и быть не доступен для нажатия.
  Шапка должна отображаться всегда в верхней части окна браузера вне зависимости от прокрутки страницы.
  Страница Главная должна содержать название вашего приложения, логотип, шапку, небольшое текстовое описание и изображение. Можно в качестве референса ориентироваться на пример представленный на рисунке 2.1.

  Рисунок 1.1 Пример компоновки главной страницы

  Для изображения рекомендуется использовать векторный формат. В растровом формате,  изображение такого разрешения будет много весить и долго подгружаться, но это будет заметно только при развертывании на сервере. При локальной отладке этого не будет заметно.
  Изображения в подобном стиле можно найти в интернете, например на сайте https://storyset.com/.

  Вместо страницы Проекты необходимо реализовать страницу заглушку с ошибкой 404. Также рекомендуется добавить изображение. Пример изображения на странице представлен на рисунке 2.2.

  Рисунок 1.2 Пример изображения для страницы 404

  Изменить отображение формы добавления новых задач.
  Использовать тег <dialog>;
  При нажатии кнопок должен происходить выход из модального окна;
  Выход должен происходить также при нажатии на область вне модального окна;
  Модальное окно необходимо спозиционировать по середине экрана;
  Вне модального окна фон должен быть затемнен.
  Для отображения списка карточек задач необходимо использовать flexbox;
  Реализовать адаптивную верстку страниц. Страницы должны корректно отображаться в десктопном варианте, на планшете и телефоне;
  Реализовать смену вида меню с панели в верхней части страницы на бургер меню с появляющимся сайдбаром поверх основного контента страницы (при отображении на планшете и телефоне);
  При нажатии на иконку бургер меню должен появляться я навигационное меню;
  Реализовать анимацию появления навигационное меню;
